# Kanji Roguelike Game Specifications

## 1. システム概要
漢字の合体を核としたデッキ構築型ローグライク。
プレイヤーは収集した漢字カードでデッキを編成し、戦闘中に漢字を合体させながら敵を倒す。

## 2. デッキ・カード循環システム
- **カード循環**: 使用したカードは「捨て札」へ移動。
- **リシャッフル**: 山札が空になった際、捨て札をシャッフルして山札に再充填する。
- **永続デッキ**: 戦闘勝利時に倒した敵の漢字カードを獲得し、プレイヤーのインベントリ（永続デッキ）に追加される。
- **戦闘開始時**: インベントリまたはデッキ管理（`DeckManager`）で設定されたカードから「山札」が構築され、5枚の手札が配られる。

## 3. 戦闘メカニクス「漢字ブレイク」
特定の条件でボーナスが発生する「1 MORE」風システム。
- **相殺 (Mirror Clash)**: 敵と同じ漢字で攻撃するとダメージ特大（3倍）＆ AP+1。
- **構成数マウント (Overpower)**: 敵より構成数（パーツ数）が多い漢字で攻撃するとダメージ1.5倍。
- **敵合体 (Enemy Fusion Break)**: 敵にカードをドラッグして合体させると、敵が変化し「スタン」状態になる ＆ AP+1。

## 4. AP（行動力）とターン進行
- **AP制限の撤廃**: 最大値の制限なく、合体やブレイクでAPを無限に蓄積可能。
- **オートターンエンド**: 
    - APが0かつ、手札で可能なアクション（カード使用・合体・敵合体）が一切なくなった場合、自動的に敵のターンへ移行する。
    - **安全装置**: ターン開始（ドロー演出中など）から1.0秒間は判定を行わず、誤ってターンが終了するのを防ぐ。
    - **フェイルセーフ**: 山札・捨て札・手札がすべて空になった場合、無限ループを防止するため強制的にターンを終了し、エラーログを出力する。
- **スタン復帰**: 敵がスタンから復帰する際、プレイヤーの手札や山札の状態を維持したままプレイヤーターンが再開される（リシャッフルは行われない）。

## 5. 演出 (VFX)
- `VFXManager`によるスローモーション、シェイク、閃光エフェクト。
- 漢字の合体による進化演出。
- ブレイク発動時の専用カットイン演出。
